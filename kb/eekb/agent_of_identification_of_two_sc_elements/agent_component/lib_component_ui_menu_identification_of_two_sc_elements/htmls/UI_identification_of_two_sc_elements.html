Команда отождествления двух sc-элементов предназначена для отождествления двух sc-элементов. Первым аргументом команды является sc-элемент, для которого мы хотим сгенерировать связки, инцидентные второму sc-элементу. Вторым аргументом команды является sc-элемент, инцидентные связки которого нужно перенести на первый sc-элемент с устранением возможного дублирования связок. Результатом выполнения команды является генерация для первого sc-элемента связок, инцидентных второму sc-элементу, и удаление второго sc-элемента.
